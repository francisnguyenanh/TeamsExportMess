📋 Manual Token (DevTools)

Copy a Bearer token from your browser's DevTools and paste it here. Use this as a fallback when SSO or password login is unavailable.

⚠️ Important: You need a token issued for Microsoft Graph, not the Teams web app. The token must come from a graph.microsoft.com network request. If the token's audience is wrong, you will get a 401 error.
▼ How to get a Graph API token from DevTools
  1. Open https://teams.microsoft.com and sign in.
  2. Press F12 → tab Network → tick Preserve log.
  3. In the filter box type exactly: graph.microsoft.com — you must see requests to that domain, not teams.microsoft.com.
  4. Click any channel or team in Teams sidebar to trigger API calls.
  5. Click any request whose URL starts with https://graph.microsoft.com/ → tab Headers → scroll to Request Headers.
  6. Find Authorization: Bearer eyJ… and copy only the part after Bearer .
  7. Paste below — the validator will confirm the audience is correct before saving.
💡 Tip: Token is valid for approximately 60–90 minutes. The tool will attempt to auto-refresh the token if it expires during an export. For SSO to work, your PC must be domain-joined and already signed in to Teams.
